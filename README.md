# databases-buffet
Project for databases discipline.


## Requisitos:
	É necessário instalar os seguintes requisitos do sistema:
		postgresql-10
		python3
		python3-psycopg2
	Obs.: O sistema foi testado apenas no ambiente Linux.

## Execução:
	python3 interface.py

## Geração e alimentação do banco de dados:
	./run.sh
